# Penta PPT Design System

> 본문(콘텐츠) 슬라이드 기준. **4p · 6p의 헤더 영역과 그 바로 아래 본문**을 실측해 정리했습니다.
> 모든 수치는 원본 `.pptx`의 XML에서 추출한 실제 값입니다. (단위: pt = 폰트/줄간격, in = 위치/크기)

---

## 1. 캔버스 / 그리드

| 항목             | 값                                                           |
| ---------------- | ------------------------------------------------------------ |
| 슬라이드 크기    | 13.333 × 7.5 in (12192000 × 6858000 EMU) — 16:9 와이드       |
| 좌측 본문 정렬선 | **x = 0.535 in** (모든 헤더·본문·콘텐츠가 이 선에 좌측 정렬) |
| 우측 여백        | 약 0.53 in (본문 폭 12.27 in 기준 좌우 대칭)                 |
| 본문 콘텐츠 폭   | 12.27 in                                                     |

좌우 여백이 약 0.53 in로 거의 대칭입니다. 새 요소는 **x = 0.535 in 정렬선**에 맞추세요.

---

## 2. 폰트

| 용도                                   | 서체                          |
| -------------------------------------- | ----------------------------- |
| 슬라이드 제목(헤더), 강조 텍스트       | **Pretendard SemiBold**       |
| 카테고리 라벨, 섹션 헤더, 소제목, 날짜 | **Pretendard Medium**         |
| 본문, 카드 본문                        | **Pretendard** (Regular)      |
| 테마 폴백(미설치 시)                   | Cambria(제목) / Calibri(본문) |

전 슬라이드가 Pretendard 패밀리로 통일되어 있습니다. Pretendard 미설치 환경 대비 폴백 지정을 권장합니다.

---

## 3. 색상 토큰

| 토큰             | HEX                   | 사용처                                                  |
| ---------------- | --------------------- | ------------------------------------------------------- |
| Text / Primary   | `#404040`             | 제목·본문 텍스트 (테마 tx1 @ 75% lum + 25% off)         |
| Label / Muted    | `#A6A6A6`             | 상단 카테고리 라벨 (테마 bg1 @ 65% lum)¹                |
| Header BG (Pill) | `#000000`             | 섹션 헤더 알약 배경                                     |
| Header Text      | `#FFFFFF`             | 알약 위 헤더 글자                                       |
| Sub-band BG      | `#64B5F6` @ alpha 10% | 소제목 밴드 (흰 배경 위에서 `#EFF5FE` 정도의 옅은 파랑) |
| Accent / Link    | `#0C73EF`             | WAF 등 핵심 키워드 하이라이트                           |
| Warning Red      | `#FF4040`             | 경고·피해 강조                                          |
| Warning Orange   | `#FF5722`             | 경고·피해 강조(보조)                                    |
| Divider Gray     | `#D2D2D2`             | 구분선·보조 도형                                        |

**테마 보조 블루 계열**: accent1 `#30A2DB` · accent2 `#1D7ED7` · accent3 `#1B5DA5` · 진한 블루 `#0060A9`

> ¹ 카테고리 라벨은 파일상 **bg1(흰색) 65% 명도 = `#A6A6A6` 회색**으로 정의되어 PowerPoint에서는 뮤트 그레이로 보입니다. (일부 뷰어/변환기에서는 테마 해석 차이로 파란 기운이 섞여 보일 수 있습니다. 브랜드 블루 라벨을 원하면 `#1B5DA5` 권장)

## 4. 헤더 영역 (페이지 상단) — 4p · 6p 공통

세 요소가 좌측 정렬선(x=0.535 in)에 위로부터 **카테고리 라벨 → 제목 → 본문** 순서로 쌓입니다.

### 4-1. 카테고리 라벨 (eyebrow)

예: `웹방화벽의 필요성`

| 속성            | 값                             |
| --------------- | ------------------------------ |
| 위치            | x = 0.535 in, **y = 0.413 in** |
| 폰트            | Pretendard Medium **10 pt**    |
| 색상            | `#A6A6A6`                      |
| 정렬            | 좌측 정렬, 세로 가운데         |
| 줄간격          | 고정 16.8 pt                   |
| 텍스트 안쪽여백 | 0 (상하좌우 모두 0)            |

### 4-2. 슬라이드 제목 (헤더)

예: `웹 공격으로 인한 개인정보 유출 피해 증가` / `보안 관련 국내 규제 강화`

| 속성            | 값                             |
| --------------- | ------------------------------ |
| 위치            | x = 0.535 in, **y = 0.695 in** |
| 폰트            | Pretendard SemiBold **14 pt**  |
| 색상            | `#404040`                      |
| 정렬            | 좌측 정렬, 세로 가운데         |
| 줄간격          | 고정 16.8 pt                   |
| 텍스트 안쪽여백 | 0                              |

### 4-3. 본문 (헤더 바로 아래)

예: `사이버 범죄가 2022년 1,142건에서 …`

| 속성      | 값                                          |
| --------- | ------------------------------------------- |
| 위치      | x = 0.535 in, **y = 1.201 in**, 폭 12.27 in |
| 폰트      | Pretendard **12 pt**                        |
| 색상      | `#404040`                                   |
| 정렬      | 좌측 정렬                                   |
| 줄간격    | **100% (단일)**                             |
| 단락 간격 | 단락 앞 **10 pt** (문단 사이 여백)          |
| 들여쓰기  | 좌측 ≈ 0.05 in, 글머리표 없음               |

## 5. 요소 간 수직 간격 (실측)

위에서 아래로 쌓이는 헤더 영역의 간격입니다.

| 구간                  | 기준선 간 거리         | 비고                                       |
| --------------------- | ---------------------- | ------------------------------------------ |
| 카테고리 라벨 → 제목  | 0.282 in (≈ 20 pt)     | 라벨과 제목이 거의 맞붙어 한 묶음처럼 보임 |
| 제목 → 본문           | **0.233 in (≈ 17 pt)** | 제목 줄간격(16.8pt)과 동일 리듬            |
| 본문 → 첫 콘텐츠 블록 | 약 0.49 in (≈ 35 pt)   | 본문과 카드/도형 영역 사이                 |

**규칙**: 헤더 묶음(라벨+제목)은 붙이고, 제목↔본문은 약 17 pt, 본문↔콘텐츠는 약 0.5 in로 띄웁니다.

---

## 6. 콘텐츠 내부 헤더 패턴 (4p·6p 카드/표 영역)

본문 아래 콘텐츠 영역에서 반복되는 헤더/소제목/본문 3단 구조입니다.

### 6-1. 섹션 헤더 — 검정 알약

예: `개인정보보호법 법규`, `웹 서버 보호 조치 법규`, `최근 3년 간 주요 개인정보 유출 사례`

| 속성            | 값                                     |
| --------------- | -------------------------------------- |
| 도형            | 둥근 사각형(roundRect), 배경 `#000000` |
| 폰트            | Pretendard Medium **11 pt**            |
| 글자색          | `#FFFFFF`                              |
| 정렬            | 가운데                                 |
| 텍스트 안쪽여백 | 0.039 in (상하좌우)                    |

### 6-2. 소제목 밴드 — 옅은 파랑

예: `개인정보의 안전성 확보조치 기준 제1조(목적)`

| 속성            | 값                                              |
| --------------- | ----------------------------------------------- |
| 배경            | `#64B5F6` @ 투명도 10% (흰 배경 위 ≈ `#EFF5FE`) |
| 폰트            | Pretendard Medium **11 pt**                     |
| 글자색          | `#404040`                                       |
| 정렬            | 가운데                                          |
| 단락 앞 간격    | 10 pt                                           |
| 텍스트 안쪽여백 | 0.039 in                                        |

### 6-3. 카드 본문 (법조문 등)

| 속성            | 값                             |
| --------------- | ------------------------------ |
| 폰트            | Pretendard **10 pt**           |
| 색상            | `#404040`                      |
| 줄간격          | 100% (단일)                    |
| 항목(단락) 간격 | 단락 앞 10 pt                  |
| 키워드 강조     | `웹방화벽(WAF)` 등 → `#0C73EF` |

### 6-4. 강조 텍스트 (피해 항목 등)

예: `기업 이미지 실추`, `손해배상 및 과징금 부담`

- Pretendard SemiBold **14 pt**, 색상 `#FF4040`(레드) / `#FF5722`(오렌지), 가운데 정렬

### 6-5. 캡션 / 날짜

- Pretendard Medium **7 pt** (날짜), 보조 설명 캡션은 Pretendard **6 pt**

---

## 7. 푸터 (전 슬라이드 공통)

- 좌하단: Penta Security 로고
- 우하단: `Confidential Internal Only` 보안등급 표기 + 페이지 번호(우측 끝 x ≈ 13.04 in)

---

## 8. 타입 스케일 요약

| 단계             | 크기  | 서체     | 용도                     |
| ---------------- | ----- | -------- | ------------------------ |
| 강조/제목        | 14 pt | SemiBold | 슬라이드 제목, 핵심 강조 |
| 섹션 헤더/소제목 | 11 pt | Medium   | 카드 헤더, 밴드 소제목   |
| 카테고리 라벨    | 10 pt | Medium   | 상단 eyebrow, 카드 본문  |
| 본문             | 12 pt | Regular  | 헤더 아래 도입 본문      |
| 카드 본문        | 10 pt | Regular  | 표/카드 내부 설명        |
| 날짜             | 7 pt  | Medium   | 출처 날짜                |
| 캡션             | 6 pt  | Regular  | 보조 설명                |

---

## 9. 적용 체크리스트

- [ ] 헤더·본문·콘텐츠 모두 **x = 0.535 in** 좌측선 정렬
- [ ] 카테고리 라벨(10pt 회색) → 제목(14pt SemiBold #404040)은 한 묶음으로 붙이기
- [ ] 제목 ↔ 본문 간격 ≈ 17 pt, 본문 ↔ 콘텐츠 ≈ 0.5 in
- [ ] 본문 줄간격 100%, 문단 사이 10 pt
- [ ] 섹션 헤더는 검정 알약 + 흰 11pt Medium 가운데정렬
- [ ] 소제목은 옅은 파랑 밴드(`#64B5F6` 10%) + 11pt Medium
- [ ] 핵심 키워드는 `#0C73EF`, 경고는 `#FF4040`/`#FF5722`
- [ ] 본문 텍스트 색은 순흑이 아닌 `#404040`
